# SUPPORT COMMERCIAL ZKTECO + LEOPARDO RH

## 1. Resume executif

Leopardo RH propose une solution moderne de pointage et de gestion RH qui combine :

- application mobile employe
- plateforme web manager / RH
- borne d entree entreprise avec visage + empreinte
- fonctionnement online et offline

L objectif est simple :

- fiabiliser le pointage
- moderniser l experience employe
- garder une solution exploitable meme sans internet stable

## 4. Parcours utilisateur

### 4.1 Parcours entreprise

1. le super admin cree la societe cliente
2. il cree le manager principal
3. le manager ou le RH cree les employes
4. chaque employe recoit une invitation email
5. l employe installe l application mobile
6. l employe active son compte
7. l employe complete son profil et soumet sa biometrie
8. le manager ou le RH approuve
9. l employe peut ensuite pointer :
   - via mobile
   - via borne d entree

### 4.2 Parcours pointage offline

1. la borne capture les pointages
2. le PC local stocke les donnees
3. l entreprise continue de travailler sans interruption
4. quand internet revient, les donnees sont synchronisees
5. les pointages deviennent visibles sur mobile et web
