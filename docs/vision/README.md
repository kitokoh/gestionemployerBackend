# Documents de vision - Leopardo RH

Ce dossier contient les documents fondateurs produits en avril 2026 qui decrivent la vision produit long-terme.

Les PDFs sont conserves **tels qu'ils ont ete rediges** a des fins d'archive et de tracabilite. La source de verite operationnelle pour l'equipe de developpement est constituee des documents courts a la racine de `docs/` :

- [`docs/APV.md`](../APV.md) - Manifeste 1 page (4 piliers + 12 Lois)
- [`docs/ROADMAP.md`](../ROADMAP.md) - Phasage MVP -> modules a la demande
- [`docs/STATUTS.md`](../STATUTS.md) et [`docs/COULEURS.md`](../COULEURS.md) - Tokens design partages Flutter <-> Tailwind

## Regle
