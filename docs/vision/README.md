# Documents de vision — Leopardo RH

Ce dossier contient les documents fondateurs produits avril 2026 qui décrivent la vision produit long-terme.

Les PDFs sont conservés **tels qu'ils ont été rédigés** à des fins d'archive et de traçabilité. La source de vérité opérationnelle pour l'équipe de développement est constituée des 3 documents courts à la racine de `docs/` :

- [`docs/APV.md`](../APV.md) — Manifeste 1 page (4 piliers + 12 Lois)
- [`docs/ROADMAP.md`](../ROADMAP.md) — Phasage MVP → modules à la demande
- [`docs/STATUTS.md`](../STATUTS.md) et [`docs/COULEURS.md`](../COULEURS.md) — Tokens design partagés Flutter ↔ Tailwind
