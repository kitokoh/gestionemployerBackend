# PROMPT P04 : MODULES GESTIONNAIRE & NOTIFICATIONS
# Dossier : /mobile

Offre aux managers la capacité de piloter leur équipe en mobilité.

### Objectifs :
1. **Dashboard Manager** : Vue en temps réel des présences (qui est là, qui est en retard).
2. **Approbations** : Écrans pour valider les congés et les avances en un clic.
3. **Notifications Push** : Intégration complète de FCM (Firebase) avec deep linking vers la ressource concernée.
4. **Annuaire** : Liste simplifiée des employés de l'équipe avec bouton d'appel direct.

Gère les états de chargement (shimmers) pour une sensation de fluidité.
