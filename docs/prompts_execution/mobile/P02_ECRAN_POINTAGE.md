# PROMPT P02 : MODULE POINTAGE (EXPÉRIENCE NATIVE)
# Dossier : /mobile

Développe le cœur de l'application : l'écran de pointage.

### Objectifs :
1. Écran d'accueil avec grand bouton de pointage (appui long 1.5s).
2. Gestion du GPS : vérifier que l'employé est dans le périmètre autorisé.
3. Intégration de la Caméra : prise de photo automatique lors du pointage (si option activée).
4. Scanner QR Code : alternative pour les sites équipés.
5. Historique mensuel des pointages.

Utilise `17_MOCK_DATA_MOBILE.md` pour simuler les retours API si nécessaire.
