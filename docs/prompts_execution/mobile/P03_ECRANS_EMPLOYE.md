# PROMPT P03 : MODULES EMPLOYÉ (CONGÉS, PAIE, TÂCHES)
# Dossier : /mobile

Implémente les fonctionnalités de self-service pour l'employé.

### Objectifs :
1. **Absences** : Liste des demandes et formulaire de soumission avec calendrier.
2. **Paie** : Liste des bulletins et visionneuse PDF intégrée (`flutter_pdfview`).
3. **Tâches** : Liste des tâches assignées, mise à jour du statut et commentaires.
4. **Profil** : Modification des infos personnelles et mot de passe.

Assure-toi que les calculs de solde de congés affichés correspondent aux données API.
