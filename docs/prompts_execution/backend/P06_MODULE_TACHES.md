# PROMPT P06 : MODULE TÂCHES & PROJETS
# Dossier : /api

Gère la réalisation des tâches et le suivi des projets.

### Objectifs :
1. CRUD pour les `Projects`.
2. CRUD pour les `Tasks` avec assignation multiple (via JSONB).
3. Système de commentaires sur les tâches avec pièces jointes.
4. Kanban simplifié pour le dashboard web.
5. Notifications push (FCM) lors de l'assignation d'une tâche.

Utilise le moteur de recherche JSON de PostgreSQL pour filtrer les tâches assignées à un employé.
