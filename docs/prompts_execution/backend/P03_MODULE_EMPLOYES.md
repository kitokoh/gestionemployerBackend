# PROMPT P03 : MODULE EMPLOYÉS & STRUCTURE RH
# Dossier : /api

Développe le CRUD complet des employés et de la structure organisationnelle.

### Objectifs :
1. CRUD pour : `Departments`, `Positions`, `Sites`, `Schedules`.
2. CRUD pour `Employees` avec génération automatique du matricule.
3. Gestion des rôles RBAC (Manager Principal, RH, etc.) selon `10_RBAC_COMPLET.md`.
4. Import CSV des employés.
5. Upload de la photo de profil (stockage par tenant).

Applique les Policies Laravel pour chaque action.
