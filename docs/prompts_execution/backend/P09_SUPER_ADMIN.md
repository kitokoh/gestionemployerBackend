# PROMPT P09 : SUPER ADMIN & DASHBOARD GESTIONNAIRE
# Dossier : /api

L'interface de pilotage de la plateforme et de chaque entreprise.

### Objectifs :
1. **Super Admin** : Gestion des entreprises, des plans, et facturation SaaS.
2. **Dashboard Web (Inertia)** : Création des vues réactives pour le gestionnaire principal.
3. Statistiques en temps réel (nombre d'employés présents, tâches en retard).
4. Paramétrage global de l'entreprise (Logo, Langue, Options pointage).

Utilise PrimeVue pour les composants UI du dashboard.
